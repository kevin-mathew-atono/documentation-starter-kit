# Last Evaluated

The most recent timestamp when a feature flag was evaluated (checked for ON/OFF state) by the SDK across any environment. Displayed as relative time with details on hover showing evaluation history, stability, and how long the current value has been returned. 'Not yet evaluated' means the flag has never been checked.
