# Slice

A defined subset within an environment for a feature flag, based on customer and/or location attributes with specific operators (is, is not, contains, is one of, is in, is not in). Each slice has an ON/OFF value. Evaluated top-to-bottom with first-match logic; default value used if no slice matches. A slice can have up to two conditions (one customer, one location).
