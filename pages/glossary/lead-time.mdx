# Lead Time

The total elapsed time from when a work item is created or requested to when it is fully delivered (moved to Done). Lead time encompasses both waiting time in the backlog and active work time in In Progress steps. It is a broader measure than cycle time, which counts only active work time. Reducing lead time is a key goal of lean and agile delivery practices.
