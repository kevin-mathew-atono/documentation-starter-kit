# Version History

A record of all changes to a story or bug's content (title, user stories, acceptance criteria, description) showing additions (green) and deletions (red strikethrough). Multiple edits within a short timeframe are grouped. Previous versions can be restored, creating a new version. AI Context has separate version history per section.
