# Active Sprint

The one sprint a Scrum team is currently executing toward a sprint goal. Only one sprint can be active per team at any given time. Active sprint items appear on the In Progress page, contribute to burndown charts, and count toward velocity calculations when the sprint is completed. Starting a new sprint without closing the previous one is not permitted.
