# Cumulative Flow Diagram

A stacked area chart showing the count of work items in each workflow state over time. The width of each band represents the number of items in that state; a widening band indicates accumulating work-in-progress. The horizontal distance between the entry and exit of a band approximates cycle time. Cumulative flow diagrams help teams identify bottlenecks and monitor flow stability.
