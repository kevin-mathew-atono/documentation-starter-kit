# Glossary

A structured, exportable collection of up to 200 key product concepts built from documentation by Product Knowledge. Each concept includes name, description, synonyms, and related concepts. Managed by Glossary Contributors and Glossary Owners with version history and conflict detection. Exported as TSV for use in AI tools.
