# A/B Test

A controlled experiment that presents two or more variants of a feature to different user groups simultaneously and measures behavioral differences. A/B tests generate evidence for product decisions by comparing metrics between a control group and one or more treatment groups. In Atono, A/B tests are enabled by using feature flags with customer-based slice targeting to route different users to different variants.
