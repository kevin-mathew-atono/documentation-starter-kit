# Slack Integration

A workspace integration connecting Atono to a Slack workspace, enabling Slack channels for teams and stories, posting updates for team changes and feature flag modifications, and user-level Slack account connection. Must be enabled by Workspace Owner or Administrator before individual users can connect.
