# Slack Integration

A workspace integration connecting Atono to a Slack workspace, enabling automated updates and communication across both platforms. Capabilities include: creating dedicated Slack channels for teams and individual stories, posting updates when team settings or feature flag configurations change, and connecting individual Atono user accounts to their Slack identities for mention routing. Configured by an Administrator from Workspace Settings.
