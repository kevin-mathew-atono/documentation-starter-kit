# Invite

The process of adding a new user to an Atono workspace by sending them an email invitation containing a join link. Invitations are sent by Workspace Owners or Administrators. The invited user must accept the invitation and create or sign in to an Atono account. Once accepted, the user is added to the workspace with the Standard User role by default.

## Also Known As

- Workspace Invite
