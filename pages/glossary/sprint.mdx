# Sprint

A time-boxed iteration for Scrum teams, typically one to four weeks, with defined start and end dates and a committed set of stories working toward a sprint goal. Each team can have one active sprint and unlimited future sprints planned. Sprint scope can be adjusted mid-sprint with a warning. Completed sprints contribute to velocity calculations. Sprint history is visible in the Sprint Calendar.
