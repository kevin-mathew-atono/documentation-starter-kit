# Flag Configuration

The settings controlling a feature flag's behavior in each environment. Includes ON/OFF state and optional slices defining conditions based on customer or location attributes. Configuration changes are versioned and appear in Activities. Slices are evaluated top-to-bottom with first-match logic.
