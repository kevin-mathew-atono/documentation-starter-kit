# Integration

A connection between Atono and an external tool or service that synchronizes data or triggers actions across systems. Integrations are configured at the workspace or team level and include GitHub for pull request linking, Slack for notifications, and identity providers for SSO. Activity originating from an integration is labeled with its source in the Activities log.

## Also Known As

- External Integration
- App Integration
