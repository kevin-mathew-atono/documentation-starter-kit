# Feature Engagement

A story-level usage graph showing how often a feature is used, filterable by environment, customer, or location. Displays top 10 customers by default or compares usage across environments. Supports bar charts (totals) and line graphs (trends). Requires SDK-based usage tracking or mapped clicks via Chrome extension. Graph position moves to top of story when in Done category.
