# Cycle Time Report

A team-specific report visualizing average cycle times for stories and bugs over a selected timeframe (1-60 months). Resolution adjusts automatically (daily, weekly, monthly, quarterly). Can be filtered by story size. Shows which data is used in cycle time calculations based on team configuration. Uses team time zone for date grouping.
