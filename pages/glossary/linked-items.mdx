# Linked Items

Stories, bugs, or epics explicitly connected to each other to show relationships like dependencies or related work. Links are bidirectional and visible in the Linked Items section of each item. Can include suggested items based on content similarity. Does not include inline item ID mentions, which are clickable but not formal links.
