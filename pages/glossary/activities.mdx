# Activities

A chronological, filterable log of all changes to a story, bug, epic, or feature flag including title changes, workflow transitions, assignments, comments, attachments, subtasks, and pull requests. Each entry shows who performed the action and when. Activities can be filtered by type or source (Atono, Integration, API, MCP Server).
