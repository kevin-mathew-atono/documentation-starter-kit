# Definition of Ready

A set of criteria a story must satisfy before a team can commit to working on it in a sprint or pull it from the backlog. Typical criteria include having clear acceptance criteria, an estimated size, and no unresolved dependencies. The definition of ready reduces mid-sprint surprises and helps teams start work with shared understanding.
