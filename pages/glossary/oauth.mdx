# OAuth

An open authorization standard that allows third-party applications to access Atono resources on behalf of a user without requiring the user to share their credentials. OAuth is used by external integrations and tools that need delegated access to workspace data. Atono supports OAuth for specific integration scenarios where API key authentication is not sufficient.
