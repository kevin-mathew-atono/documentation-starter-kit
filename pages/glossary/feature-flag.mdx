# Feature Flag

A named toggle that controls whether a feature is active in a given environment without requiring a code deployment. Each flag has a unique key, an optional description, a flag owner, and per-environment on/off states. Flags can target specific customers via slices. They can be created standalone or linked to a story. The Atono SDK evaluates flags at runtime and records evaluation timestamps for observability.
