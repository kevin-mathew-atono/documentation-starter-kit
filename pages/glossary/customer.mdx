# Customer

A named tenant or organization whose users interact with your product, identified by a string value passed to the Atono SDK or Chrome extension. Used to scope feature flag slices to specific customers and to filter Feature Engagement graphs. Customer values appear in Slice conditions, usage tracking events, and Engagement page filters.
