# Webhook

An HTTP callback that sends real-time event data from Atono to an external URL when a specified event occurs, such as a story being created, updated, or moved to a new workflow step. Webhooks enable integrations with external systems like CI/CD pipelines, dashboards, or custom automation tools without requiring the external system to poll Atono continuously.
