# Item ID

A unique identifier assigned to each story, bug, or epic in the format TYPE-NUMBER (e.g., STORY-123, BUG-456, EPIC-789). Assigned sequentially; numbers are not reused unless the deleted item was the next available ID. Used for linking, searching, and referencing items.
