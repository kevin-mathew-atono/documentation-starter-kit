# Epic

A planning item that groups related stories under a single shared goal, providing a high-level view of progress across multiple pieces of work. Epics include a title, one or more user stories defining intended outcomes, and a description. Stories can belong to only one epic at a time. When a story is split, both resulting stories remain in the original epic unless a new epic is created during the split. Epics appear in search results, the Everything page, and timelines.
