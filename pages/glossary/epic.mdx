# Epic

A planning item that groups up to 100 related stories under a shared goal. Includes title, one or more user stories defining intended outcomes, description, and progress tracking. Stories can belong to only one epic. When splitting stories, both resulting stories remain in the same epic, and a new epic can be created during the split if needed. Epics appear in search results and the Everything page.
