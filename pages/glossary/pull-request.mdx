# Pull Request

A proposed code change in Git-based version control systems, automatically linked to stories or bugs when mentioned by ID in the PR title, description, branch name, or optionally comments and commit messages. Linked PRs appear in Atono with details (title, number, status, assignee, build status) and are logged in Activities. Requires GitHub integration.
