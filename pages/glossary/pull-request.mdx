# Pull Request

A proposed code change in a Git-based version control system, automatically linked to Atono stories or bugs when the Atono item ID is mentioned in the PR title, description, or branch name. With the GitHub integration, PR mentions in comments and commit messages can also be configured to auto-link. Linked pull requests appear in the story or bug's Activities and in the Linked Items section, showing status (open, merged, closed) and a direct link to the PR.
