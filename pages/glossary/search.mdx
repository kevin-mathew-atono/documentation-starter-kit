# Search

A global fuzzy keyword search across all stories, bugs, and epics the user has permission to view. Search matches against titles, item IDs, descriptions, user stories, acceptance criteria, and active (unresolved) comments. Results are ranked by relevance. Search is accessible from the top navigation bar and can be used alongside filters on the Everything page for more precise queries.
