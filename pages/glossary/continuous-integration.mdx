# Continuous Integration

A software development practice where team members merge code changes into a shared branch frequently, triggering automated builds and tests on each merge. Continuous integration surfaces integration errors quickly and reduces the risk of large, difficult merges. In Atono, pull requests linked to stories provide visibility into CI status as part of the development workflow.
