# Atono MCP Server

A local Docker-based server implementing the Model Context Protocol that connects AI development tools like Cursor, Claude, or VS Code to an Atono workspace. Allows AI tools to read and update stories, bugs, workflows, and timeboxes using authenticated workspace access. Updates are marked with a diamond icon in Activities.
