# Bug

An error, flaw, or unintended behavior in software requiring resolution. Includes title, description with template sections (summary, reproduction steps, expected behavior, actual behavior), risk rating (probability × impact, 1-25 scale), affected environments, reporter, and optional attachments. Moves through team workflows and Bug Triage. Supports cycle time tracking, duplication, version history, comments, and linked items.
