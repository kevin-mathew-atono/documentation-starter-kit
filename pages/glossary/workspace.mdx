# Workspace

The top-level container for all teams, stories, bugs, users, and other elements in Atono for a company. Each workspace has a unique URL (example.atono.io), name, up to 50 teams, and plan-based limits for users and storage. Created during initial signup with the creator as Workspace Owner.
