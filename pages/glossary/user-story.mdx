# User Story

A brief, structured description of a feature from an end user's perspective, written in the format: 'As a [persona], I want to [action] so that [benefit].' Each story or epic can contain one or more user stories. The persona identifies the type of user; the action describes the desired capability; the benefit explains the underlying motivation or outcome. User stories guide acceptance criteria authoring and help teams stay focused on user value.
