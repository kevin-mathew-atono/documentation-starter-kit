# User Story

A brief description of a feature from the user's perspective in the format 'As a [persona], I want to [action] so that [benefit].' Stories can have multiple user story statements. Atono prompts for all parts and suggests previously-used personas. Epic-level user stories describe higher-level intent across an entire initiative.
