# Comment

Feedback or discussion added to a story or bug, either generally or on specific highlighted text or acceptance criteria. Supports @mentions for notifications, inline item ID linking, clickable URLs with highlighting, and actions (accept, reject, edit, delete). Active comments appear in a dedicated panel; addressed comments move to Activities.
