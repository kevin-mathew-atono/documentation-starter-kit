# Flag Evaluation

The process by which the Atono SDK determines whether a feature flag is ON or OFF for a given user, customer, or location at runtime. During evaluation, the SDK checks the flag's per-environment configuration and applies any defined slices to return a boolean result. The timestamp of the most recent evaluation is recorded and visible in the flag's status panel.
