# Filter

Criteria applied to narrow the set of visible items in backlogs, the Everything page, timelines, or reports. Filters can target fields including assignee, team, product theme, story size, workflow step, epic, sprint, timebox, risk rating, and more. Multiple filters combine with AND logic. Active filters persist within a session and can be saved as a Saved View for future reuse.
