# Keyboard Shortcut

A key or combination of keys that triggers a specific action in Atono without using the mouse. Keyboard shortcuts speed up common operations like opening search, creating a new story, navigating between pages, and submitting forms. A full list of shortcuts is available from the help menu within the application.

## Also Known As

- Hotkey
- Key Binding
