# Flag Targeting

Rules within a flag's per-environment configuration that determine which customers or locations see the flag as enabled. Targeting uses slice definitions with operators (is, is not, contains, is one of, is in, is not in) to evaluate customer identifiers and location data. When no targeting rules match, the flag falls back to the environment's default ON/OFF state.
