# Bulk Edit

Simultaneously applying a change to multiple selected stories or bugs at once. Supported bulk edits in Atono include updating the assignee, workflow step, story size, product theme, timebox, and team. Bulk editing reduces repetitive work when reorganizing a backlog or updating a set of items after a scope decision.
