# Story Map

A two-dimensional visual arrangement of user stories organized by user journey across the horizontal axis and priority or detail level down the vertical axis. Story maps help teams see the complete user experience, identify gaps, and plan incremental releases. The top row represents the high-level journey steps (backbone); lower rows contain the detailed stories that support each step.
