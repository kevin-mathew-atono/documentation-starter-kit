# Mention

Tagging a specific user in a comment or description by typing @ followed by their name or email. A mention sends the tagged user a notification and creates a link to their profile. Mentions are supported in story and bug comments and in the description fields. Mentioned users are notified by email.

## Also Known As

- "@mention, Tag"
