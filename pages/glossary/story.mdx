# Story

The primary unit of user-focused work in Atono. Each story has a title, one or more user stories in 'As a / I want to / So that' format, acceptance criteria, story points, an assignee, an epic, and optional linked feature flags. Stories move through a team's workflow, support comments, attachments, subtasks, and version history, and can be duplicated or split into multiple stories.
