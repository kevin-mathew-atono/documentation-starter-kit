# GitHub Integration

A workspace integration connecting Atono to a GitHub organization, enabling automatic linking of pull requests to stories and bugs. Pull requests are linked when the story or bug ID appears in the PR title, description, branch name, or optionally in comments and commit messages. Requires a Workspace Owner or Administrator to configure.
