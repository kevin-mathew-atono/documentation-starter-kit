# Private Team

A team with restricted visibility where only team members, Team Administrators, Backlog Owners, and Workspace Owners/Administrators can see the team's backlog and work items. Non-members see the team name but not its contents. Team members of a private team can manage membership. Contrast with public teams, which any workspace user can access.
