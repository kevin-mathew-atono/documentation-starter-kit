# Dependency

A relationship where one work item cannot be started or completed until another item is finished. Dependencies create sequencing constraints and are a common source of sprint planning risk. Identifying dependencies early helps teams reorder work, negotiate delivery, or split stories to reduce blocking chains. In Atono, dependencies can be represented using linked items.

- Blocker Dependency
