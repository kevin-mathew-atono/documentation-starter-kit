# Flag Permissions

Access controls for feature flags, separate from workspace and team roles. Flag User can modify configuration and toggle flags. Flag Owner has User permissions plus ability to edit name/description, manage permissions, remove from stories, and delete flags. Creator is automatically assigned Owner. Teams are auto-assigned User permissions when a flagged story is added to their backlog if no team currently has permissions.
