# API Endpoint

A specific URL path in the Atono public API representing a resource or operation, such as listing stories, creating a bug, or fetching a workspace's environments. Each endpoint accepts defined HTTP methods (GET, POST, PATCH, DELETE) and requires authentication via an API key. Endpoint documentation is available in the Atono API reference.
