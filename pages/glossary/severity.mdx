# Severity

The degree of impact a bug has on users or core system functionality, independent of how likely users are to encounter it. Severity is distinct from probability: a crash on a rare code path is high severity but low probability. In Atono, severity is captured as part of the impact field during bug triage, contributing to the overall risk rating.
