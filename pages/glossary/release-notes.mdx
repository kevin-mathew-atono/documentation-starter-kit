# Release Notes

A document summarizing the new features, improvements, and bug fixes included in a product release, typically published when a release goes live. Release notes help users and stakeholders understand what has changed and why, reducing support burden and setting expectations. In Atono, the content for release notes is often sourced from completed stories and bugs linked to a specific release or timebox, using the release description and story acceptance criteria as the source of truth.
