# Usage Tracking

Recording user interactions with features to power Feature Engagement graphs. Implemented by calling SDK record() methods in code or by mapping clicks via the Atono Chrome extension. Requires SDK v1.1.0+ running client-side. Events include story ID, environment, optional customer, and auto-detected location from IP.
