# SSO Mode

A workspace setting controlling how SSO is applied: On (all users use SSO), Off (all users use email login), or Testing (only Workspace Owners and Administrators use SSO). Changing modes may invalidate existing sessions. Set during initial SSO configuration or updated anytime by Workspace Owners/Administrators.
